macOS Utility

AxiomOS Companion Setup

Integrate the Swift daemon into your macOS desktop to unlock global shortcuts, direct window focus context interception, and background API routing. Compiling from source provides absolute security transparency.

1
Clone the Repository
Open your Terminal application and clone the codebase repository directly to your machine:
git clone https://github.com/Sreeram5678/Axiom.git
2
Compile Release Swift Bundle
Navigate into the cloned directory and invoke the native Swift Package Manager compiler to build the optimized release package:
cd Axiom/AxiomOS && swift build -c release
3
Deploy App Container to /Applications
Copy the compiled application bundle into your system applications deck:
cp -R .build/release/AxiomOS.app /Applications/
Security Benefit Zero Gatekeeper Warnings

Frictionless Installation: Because you are compiling and launching AxiomOS natively from source directly on your own hardware, macOS automatically stamps it as a trusted developer binary. The application will open instantly with zero Gatekeeper quarantine warning blocks or security blocks!

Browser Integration

Axiom Chrome Extension Setup

Load the structural client extension in Google Chrome to enable in-page DOM parsing, schema injections, and WebCrypto settings management.

1
Load Unpacked Developer Module
Navigate to chrome://extensions/ in Google Chrome. Toggle the Developer mode switch in the upper-right corner of the pane to active. Click the Load unpacked button in the action bar, and choose the root directory of your cloned Axiom codebase folder.
2
Insert Gemini API Credentials
Click the Axiom extension icon in your Chrome toolbar. Tap the settings cog icon at the base of the viewport panel, paste your private Gemini Developer Key (generated securely via Google AI Studio), and hit Save Settings. Sync keys are immediately isolated using AES-GCM local storage.

System Security

Accessibility Trust Integration

Because AxiomOS registers system-wide shortcuts and performs in-place text replacements, macOS requires accessibility privileges to handle event writes securely.

1
Accessibility Dialog Overlay
Upon triggering the HUD panel for the first time via Control+Shift+Space, macOS automatically locks the selection and prompts a standard "Accessibility Access" approval modal.
2
System Panel Authorization
Navigate to System Settings > Privacy & Security > Accessibility, select AxiomOS in the app list (or authorize the launching Terminal shell if compiling via CLI), and switch the toggle switch to ON.
Keychain Storage

Interactive Configuration Builder

Configure your native settings in real-time. AxiomOS securely persists these settings at ~/.axiom_config.json, auto-migrating sensitive keys to macOS Keychain buffers.

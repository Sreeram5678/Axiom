## 🚫 Rejected Ideas
*Features or architectures evaluated and rejected to maintain performance, security, or design principles.*

- **Electron-based Desktop Port**: Rejected using Electron/JS wrapper for the macOS app in favor of a **100% native Swift/AppKit/SwiftUI** accessory application. This keeps the application extremely lightweight (~30MB RAM idle, 0.0% CPU) and allows seamless OS-level integrations.
- **Plaintext API Key Storage**: Rejected storing API credentials in unencrypted config files or raw local storage. Implemented **macOS Keychain Services** for AxiomOS and **AES-GCM 256-bit PBKDF2 WebCrypto encryption** for the Chrome Extension.
- **Polling-based Streaming Updates**: Rejected polling APIs for streaming completions. Designed a **custom line-buffered SSE (Server-Sent Events) parser** to achieve smooth, real-time word-by-word injection in target editors.

## 🟢 Approved Ideas
*Core architecture and feature sets built, validated, and approved in the final stable release.*

### 🖥️ AxiomOS (macOS Native Companion)
- **Interactive HUD Menu**: A gorgeous SwiftUI-based glassmorphic panel overlaying system-wide under the mouse cursor. Supports arrow-key selection, character chords, and real-time streaming preview.
- **Carbon Global Hotkeys**: Deep integration with Carbon Event Manager system hooks (`Control+Shift+Space` for HUD, and direct shortcuts like `Control+Shift+O/P/R/S/E`).
- **Direct-to-Editor Streaming Replacement**: Employs macOS Accessibility (`AXUIElement`) APIs to inject optimized text in-place into any active text editor in real-time.
- **Reliable Fallback Sequence**: Asynchronous, non-blocking clipboard restoration (`Cmd+C` / `Cmd+V`) sequence if target applications are sandboxed or untrusted.
- **Single-Instance Lock Prevention**: Native locking mechanism preventing duplicate agent instances, complete with user notifications via `UNUserNotificationCenter`.
- **Dynamic Config Manager**: Fast, non-volatile configuration parsing supporting Gemini model switching, custom system personas, and segmented output length limits (Short, Medium, Long).
- **Standalone App Bundle Builder**: Custom packaging script (`build_app.sh`) generating correct icons, handling Swift SPM assets, and building a fully standalone `AxiomOS.app`.

### 🌐 Axiom Chrome Extension
- **Inline Optimization Capsules**: Floatable "✨ Optimize Prompt" triggers dynamically injected directly over input fields of ChatGPT, Gemini, Claude, DeepSeek, and Google AI Studio.
- **Hybrid Intelligent Routing**: Automatically checks for `window.ai` capability to route optimizations to local **Gemini Nano** (on-device AI) to minimize latency, falling back gracefully to the Gemini Cloud API.
- **Zero-Knowledge Settings Sync**: AES-GCM 256-bit settings encryption with PBKDF2 iterations using standard WebCrypto APIs before syncing encrypted settings payload via Chrome Sync.
- **Modular Persona Configurator**: Config options page with live JSON schema validation allowing developers to custom-tune LLM prompt personas.

### 🧪 Quality Assurance & CI/CD
- **Native Test Suites**: Dedicated Xcode-compliant XCTest target validating configurations and secure Keychain logic.
- **Extension Spec Coverage**: Vitest framework integration covering core crypto, API handlers, and storage sync modes.
- **Automated Workflows**: Multi-job GitHub Actions runner verifying browser extensions and compilation of macOS native targets.

## 📝 Retrospective & Implementation Notes

### Key Wins
1. **Performance**: Leveraging AppKit and Carbon hooks achieves nearly 0ms hotkey trigger lag. The line-buffered parser totally resolved UI thread stuttering during high-frequency token generation.
2. **Security & Trust**: Building client-side encryption layers inside the extension and using system Keychains ensures enterprise-grade security compliance.
3. **Refined UX**: Automatic, asynchronous clipboard restoration and native launch agents deliver a premium, native-feeling macOS experience.

### Next Steps & Future Scope
- **Universal Local Routing**: Expand the desktop companion to optionally connect with local LLM runtimes (like Ollama or Apple MLX) when offline.
- **Advanced Context Injection**: Implement native context-awareness to inject recent directory layout or git diffs automatically into prompts.
